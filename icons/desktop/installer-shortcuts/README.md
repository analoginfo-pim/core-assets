# Installer shortcut icons (MSI desktop / Start Menu)

Distinct multi-resolution `.ico` files for AIC Offline MSI shortcuts. WiX must **never** use `Icon/@SourceFile` on a configurator `.exe` (Tauri / WebView2 shells produce corrupted static-noise desktop icons).

Generated by `scripts/generate-installer-shortcut-icons.ps1` (Python 3 + Pillow). ImageMagick `magick` is optional; this pipeline uses Pillow only. If you prefer ImageMagick, render PNG frames with `magick convert` and assemble with `magick convert frame-*.png out.ico`.

## Palette and badges

| File | Role | Palette | Badge / motif |
| ---- | ---- | ------- | ------------- |
| `aic-client-product.ico` | Agent MSI product icon (ARP) | Client teal/cyan accent | None |
| `aic-client-admin-win32.ico` | Agent configurator Win32 shortcuts | Client | **W** |
| `aic-client-admin-tauri.ico` | Agent configurator Tauri shortcuts | Client | **T** |
| `aic-client-elevate-win32.ico` | Agent elevation Win32 shortcuts | Client | Key motif + **W** |
| `aic-client-elevate-tauri.ico` | Agent elevation Tauri shortcuts | Client | Key motif + **T** |
| `aic-server-product.ico` | Server MSI product icon (ARP) | Server navy/orange accent | None |
| `aic-server-admin-win32.ico` | Server configurator Win32 shortcuts | Server | **W** |
| `aic-server-admin-tauri.ico` | Server configurator Tauri shortcuts | Server | **T** |

Each file embeds sizes **16, 24, 32, 48, 64, 128, 256** (valid ICO, not a renamed PNG).

## Regenerate

```powershell
cd c:\analog-pim\core-assets
powershell -ExecutionPolicy Bypass -File scripts\generate-installer-shortcut-icons.ps1
powershell -ExecutionPolicy Bypass -File scripts\sync-to-projects.ps1
```

Then rebuild MSIs from `pim-installers/scripts/` (they stage via `Stage-InstallerShortcutIcons.ps1`).
